Description
Network Monitor Pro is a high-precision desktop application developed for continuous network latency monitoring (ICMP Ping). Ideal for sysadmins, gamers, or power users who need to diagnose the stability of local and remote connections in real-time.

✨ Key Features
📊 Real-Time Charts (Step Plot): Square-wave visualization for micro-jitter identification.

🕒 Smart Time Axis: Real-time timestamps (HH:MM:SS) on the X-axis.

🛡️ Interface Resilience: Stable monitoring during network switches (Ethernet to Wi-Fi).

⚙️ Dynamic Adjustments: Customizable ping intervals and graph window size.

📝 Outage Logging: Records "DOWN" and "UP" events in log_quedas.txt.

🌡️ Live Statistics: Min, Max, and Average (Avg) latency tracking.

🚀 How to Run
Install dependencies:
pip install customtkinter matplotlib ping3 speedtest-cli

Execute the app:
python coração.py

Descrição
O Network Monitor Pro é uma aplicação desktop de alta precisão desenvolvida para monitoramento contínuo de latência de rede (ICMP Ping). Ideal para administradores de sistemas, gamers ou usuários que precisam diagnosticar a estabilidade de conexões locais e remotas em tempo real.

✨ Funcionalidades Principais
📊 Gráficos em Tempo Real (Step Plot): Visualização em "degrau" para identificar oscilações milimétricas.

🕒 Eixo Temporal Inteligente: Marcação de tempo real (HH:MM:SS) no eixo X.

🛡️ Resiliência de Interface: Monitoramento estável durante a troca entre Cabo e Wi-Fi.

⚙️ Ajustes Dinâmicos: Alteração da frequência de ping e janela de tempo do gráfico.

📝 Log de Quedas: Registro automático de quedas no arquivo log_quedas.txt.

🌡️ Estatísticas: Cálculo de Latência Mínima, Máxima e Média (Avg).

🚀 Como Executar
Instale as dependências:
pip install customtkinter matplotlib ping3 speedtest-cli

Execute a aplicação:
python coração.py
